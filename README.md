## Publicación

Sube estos archivos directamente al repositorio `flagfootballsala.github.io` y activa GitHub Pages desde `Settings > Pages`.

No lo publiques dentro de otro repositorio tipo `arbitro-fut.github.io/flagfootballsala.github.io`, porque eso genera rutas incorrectas y abre otro proyecto.
